## Tech Stack

**Frontend**: React 19, TanStack Start/Router/Query, MapLibre GL, Zustand, Shadcn/ui, Tailwind CSS, Vite

**Backend**: ASP.NET Core / .NET 10, ABP Framework 10, PostgreSQL, OpenIddict (OIDC)
