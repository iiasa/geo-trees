# Conventions

> For comprehensive detail on file naming, see [`frontend/docs/FILE_NAMING_CONVENTIONS.md`](../../frontend/docs/FILE_NAMING_CONVENTIONS.md).

## File Naming (kebab-case)

All `.ts` and `.tsx` source files use **kebab-case**. This is enforced automatically by `scripts/check-file-naming.js`, which runs as part of `pnpm lint`.

| File type | Convention | Example |
|-----------|-----------|---------|
| Components | `kebab-case.tsx` | `user-form.tsx` not `UserForm.tsx` |
| Hooks | `kebab-case.ts` | `use-auth.ts` not `useAuth.ts` |
| Stores | `kebab-case.ts` | `user-form-store.ts` not `userFormStore.ts` |
| Utilities | `kebab-case.ts` | `permission-utils.ts` not `permissionUtils.ts` |
| Constants | always `constants.ts` | `constants.ts` not `app-constants.ts` |
| Routes | kebab-case, dots allowed | `auth.login.ts`, `api.proxy.$.ts` |
| Route specials | exact names | `__root.tsx`, `index.tsx` |

Violations fail the build and list each offending file with the expected format.

## File Size Limit

Maximum **500 lines** per `.ts` / `.tsx` file, enforced by `scripts/check-file-size.js` (also runs during `pnpm lint`).

When a file approaches this limit, split it — for example, extract a sub-component or move logic into a dedicated hook.

**Excluded from the limit:** auto-generated files (`src/infrastructure/api/`, `*.gen.ts`, `*.gen.tsx`).

## Formatting (Biome)

- **Indentation**: tabs (not spaces)
- **Quotes**: double quotes for strings
- **Imports**: auto-organized by Biome

Run `pnpm lint:fix` or `pnpm format` to auto-apply all formatting fixes.

## Import Path Aliases

Always use path aliases rather than relative paths:

```typescript
// Good
import { Button } from "@/shared/components/ui/button";
import { useUsers } from "@/features/users/hooks/use-users";
import { userGetListOptions } from "@/infrastructure/api/@tanstack/react-query.gen";

// Avoid
import { Button } from "../../../shared/components/ui/button";
```

| Alias | Resolves to |
|-------|------------|
| `@/*` | `src/*` |
| `@/shared/*` | `src/shared/*` |
| `@/features/*` | `src/features/*` |
| `@/infrastructure/*` | `src/infrastructure/*` |

## Generated Files (do not edit manually)

These files are overwritten automatically and must never be edited by hand:

| File / Directory | Regenerated by |
|-----------------|---------------|
| `src/infrastructure/api/` | `pnpm generate-api` (fetches OpenAPI spec from backend) |
| `src/routeTree.gen.ts` | TanStack Router dev server / build |

If you need to change the API client, update the backend and run `pnpm generate-api`. If you need to change routing, add or rename files in `src/routes/`.
